# Amazon Music · Adaptive UI — DERBC Studio

Live prototype for the CMU MHCI 2026 capstone with Amazon Music.

- **`/`** — the full client in an iPhone frame (best on a laptop; on a phone the frame drops away). Tap around: Home, Find, Library, Maestro, and the **Olivia Rodrigo artist page** (adaptive profile).
- **`/adaptive-artist-v3.html`** — the artist profile standalone, with the moderator console for scrubbing listener states (`?state=cold|casual|regular|superfan`, `&lens=1` for the annotation overlay).
